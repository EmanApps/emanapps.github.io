Häufige Fragen

Werden meine Daten in der Cloud gespeichert?
Nein. Alle Daten bleiben ausschließlich lokal auf Ihrem Gerät. Es findet keine Übertragung an externe Server statt.

Was passiert mit meinen Daten, wenn ich die App lösche?
Beim Löschen der App werden alle gespeicherten Daten und Anhänge unwiderruflich entfernt. Erstellen Sie vorher ein verschlüsseltes Backup über die Export-Funktion in den Einstellungen.

Kann ich meine Daten auf ein neues iPhone übertragen?
Ja, über die Backup-Export-Funktion in den Einstellungen. Exportieren Sie Ihre Daten als verschlüsselte Datei und importieren Sie diese auf dem neuen Gerät.

Warum bekomme ich keine Erinnerungen?
Prüfen Sie unter iOS-Einstellungen → Mitteilungen → myTermine, ob Benachrichtigungen aktiviert sind. Öffnen Sie außerdem die App gelegentlich, damit Erinnerungen aktualisiert werden können.
